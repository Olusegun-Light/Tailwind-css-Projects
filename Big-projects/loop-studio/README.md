# Loopstudios Website

Loopstudios landing page from [Frontend Mentor Challenge](https://www.frontendmentor.io/challenges/loopstudios-landing-page-N88J5Onjw)

## Usage

Install dependencies

```
npm Install
```

Run Tailwind CLI in watch mode

```
npm run watch
```
To build once, run
```
npm run build
```
