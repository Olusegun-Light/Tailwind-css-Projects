# Shortly Website

Shortly landing page from [Frontend Mentor Challenge](https://www.frontendmentor.io/challenges/url-shortening-api-landing-page-2ce3ob-G)

## Usage

Install dependencies

```
npm Install
```

Run Tailwind CLI in watch mode

```
npm run watch
```
To build once, run
```
npm run build
```
